# "pet-box"
Фронт-кусочек, отработка некоторых знаний и подготовка к переносу на фулстек
### Фронт на Vue3 c примнением:

- Vue Router
- Pinia
- Moment Js
- TailwindCss

Стили написаны как на ванильном CSS, так и на TailwindCss.
